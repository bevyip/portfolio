Hi, I'm Beverly!

Alumnus of Purdue University Computer Science (Machine Learning).

Developer

I began coding my freshman year of college in National University of Singapore as a Math major. I then realized the endless possibilities and significance behind the innovations I could create by being a programmer. I decided to transfer to Purdue University in 2016 and then majored in Computer Science, specializing in Machine Learning. Since then, I have learned a variety of programming languages and enjoy developing projects.

Designer

Since young, I’ve always had an eye for design and an interest in art. When I discovered the possibilities from combining programming and visual art, I realized the importance of user-empathy and design for the front-end aspects of every application. I actively engaged in creating content for the marketing departments for several organizations when I was a student on campus and still do through my Instagram for companies around the world.

Innovator

After becoming a Computer Science major, I've begun to enjoy producing side programming projects in my free time. Ideally, I create to benefit as much of society as possible and to make an impactful difference to those in need.

Leader/ Teacher

I am an individual who pushes herself out of her comfort zone to explore and learn every day. I’ve taken up minor and major roles in several minority organizations on campus and love to meet individuals from all walks of life. This is why I pursued an Anthropology minor, a way to understand cultures other than my own. I also served as a Teaching Assistant in the CS180 course and a mentor in many CS minority programs.

WanderList

Created in a 5-day-long hackathon during my Summer at Make School 2018, this web app makes sharing trip itineraries with selected group of friends possible. Utilizing Google Maps API and MongoDB, WanderList gives the user opportunities to create, share and delete both private and shared trips and events on a single platform.
Find the code here!

WeatherWear

This web app allows users to keep track of previous weather forecasts and outfits worn. By comparing today's weather forecast pulled from Dark Sky API and Google Maps API with previous logs stored using MongoDB, the user is able to make better outfit decisions and dress appropriately for the weather. Find the code here!

My Calendar

This is a simple web app created as a final project for a systems programming class at Purdue University. It allows users to create, share, edit and delete events that automatically sync up with any personal calendar using Google Calendar API.

Jeffervescence

This web app was created during my Summer at Xtern Bootcamp 2017. It extends from a simple list app with additional features such as saving, favoriting, deleting and editing list items and utilizes localStorage, JSON and React. Find the code here!

Word Game

I wrote this Java-focused game during an Object-Oriented Programming class. It uses Java GUI to display a server-client interactive multiplayer game that allows a group of friends to compete with one another to come up with the highest number of correct terms that match randomly-generated collective nouns.

Noteheader

Inspired by Evernote, this is a note-taking web app created with React and utilizes localStorage to save, edit, create and delete notes created by the user. Besides being able to authenticate with the help of Firebase, the user can opt to also sign in through Facebook, Github, Email and more.
Find the code here!
